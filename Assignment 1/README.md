Build Machine Learning Project 

This project is about building a machine learning model that can predict weather conditions. It classifies weather into seven categories: Cloudy, Foggy, Partly Cloudy, Rainy, Stormy, Sunny, and Windy. The main goal is to see if a model can learn patterns from data and correctly identify the type of weather.

The problem we tried to solve is multi-class weather classification, where many weather types look very similar to each other. For example, Cloudy, Foggy, and Partly Cloudy often look almost the same, which makes it difficult for a model to distinguish between them. The model was trained on labeled data and tested using standard evaluation metrics like accuracy, precision, recall, and F1-score.

One interesting insight from this project is that the model performed almost equally across all classes, but the overall accuracy was still very low (around 14%). This shows that the model is not clearly able to separate different weather conditions and is close to random guessing.

One surprising thing I learned is how hard it is for a machine learning model to differentiate between similar-looking categories, even when the dataset is balanced. It made me realize that having good features is more important than just having a good model.

Overall, this project helped me understand how weather classification works and how important data quality and feature selection are in machine learning.
