🚀 Image Collection & Basic Processing
As part of my Machine Learning and Computer Vision learning journey, I recently completed a hands-on project focused on building and preparing an image dataset from scratch.

📌 What I Worked On
● Collected images from Google/bing using the Python icrawler library
● Organized images into category-wise folders
● Verified image counts and ensured dataset quality
● Removed corrupt and unreadable image files
● Explored image properties such as dimensions and color channels
● Displayed sample images for visual inspection
● Converted images to RGB format and resized them to 128×128 pixels
● Transformed images into NumPy arrays and flattened them into feature vectors
● Created a Pandas DataFrame containing image data and labels

💡 Interesting Insight
While exploring the dataset, I discovered that images from the same category often had completely different resolutions, aspect ratios, and color formats. This highlighted the importance of preprocessing before using image data for machine learning models.

🎯 What I Learned
One of the biggest lessons from this project was understanding how much effort goes into preparing data before model training begins. Data collection, cleaning, and preprocessing are critical steps that directly impact the quality and performance of machine learning models.
